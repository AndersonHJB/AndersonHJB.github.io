    <table> 标签定义 HTML 表格。

    简单的 HTML 表格由 table 元素以及一个或多个 tr、th 或 td 元素组成。

    tr 元素定义表格行，th 元素定义表头，td 元素定义表格单元（列）。

    eg:

    #武汉疫情明星捐款# 都说正能量的偶像影响正能量的粉丝，在这次疫情中，肖战的“小飞侠”们为武汉捐款捐物。<br />
