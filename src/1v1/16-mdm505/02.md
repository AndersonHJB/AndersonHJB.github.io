## 1. 数字型

![image-20230110140027208](./02.assets/image-20230110140027208.png)

```python
In [1]: 1 + 1
Out[1]: 2

In [2]: 1 + 1.0
Out[2]: 2.0

In [3]: 3 - 1
Out[3]: 2

In [4]: 3 - 1.0
Out[4]: 2.0

In [5]: 4.1 * 2
Out[5]: 8.2

In [6]: 4 * 2
Out[6]: 8

In [7]: 9 / 3
Out[7]: 3.0
```

::: tip 总结

1. 如果，其中有一个数据类型是小数的话，最后的结果就是浮点数「优先级最高」
2. 除法涉及精度问题，所以最后是浮点数。

:::

def _split(num):
    if len(num) != 2:
        print("no")
        return 0
    else:
        # num = str(num)
        a, b = num[0], num[1]

        return a, b


if __name__ == "__main__":
    a = int(input())
    w1 = _split(a)
    print(w1)
    print(w1[0])
