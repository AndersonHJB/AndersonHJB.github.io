## 1. 理解变量

### 1.1 变量的含义

- 变量：
    - 变：变化
    - 量：大小
- 信封是不是在我们当前所处的空间，开辟了一个：信封✉️空间，这个空间可以存放东西「数据」。
- 冰箱：在我当前所处的空间，开辟一个空间，叫：冰箱。冰箱可以储存东西「数据」。
- <span style="color: Orange">**变量，就是在计算机的内存上，开辟空间。来存储数据。**</span>

### 1.2 变量的特点

相同的变量名「同一个信封」，数据会被覆盖。

## 3. 变量的实操

```python
x = 1
x = x + 10
print(x)

name1 = "lilei"
name2 = name1
print(name2)

name1 = "小明"
name1 = "ai"
print(name1)
```

输出：

```python
11
lilei
ai
```

::: tip 注释

注释，就是在代码最前面加 `#`，这样我们人可以看见注释内容，计算机看不见。快捷键：ctrl + /

:::

```python
# x = 1
# x = x + 10
# print(x)

# name1 = "lilei"
# name2 = name1
# print(name2)

name1 = "小明"
name1 = "ai"
print(name1)
```

输出：

```python
ai
```
