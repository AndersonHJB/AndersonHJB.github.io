## 1. 数字型「int、float」

```python
a = 1
t = type(a)
print(t)

print(type(a))

# ------ output ------
<class 'int'>
<class 'int'>
```

```python
a = 1.1
t = type(a)
print(t)

print(type(a))

# ------ output ------
<class 'float'>
<class 'float'>
```

## 5. 字典「dict」

:::: tabs

@tab Code

```python
d = {1: "one", 2: "two"}
print(type(d))
```

@tab 特性

### 1. 定义

字典是由一系列的 key 和 value 组成的，`d = {key1: value1, key2: value2}`

### 2. key 的特点

字典的 key 要求是不可变的，可变的不能当作 key，列表、集合、字典不能当作字典的 key。

### 3. Value 的特点

任意数据类型。

### 4. 可变性

字典是可变的，可以添加、修改、删除数据。

### 5. 字典的“有序性”

字典在 Python 3.6+ 之后变成有序性，但是 3.6 之前不能保证有序性，该有序性指的是：**字典底层实现是有序性的，不是前面传统意义上的有序性。**

::: tip 提示

这个有序性，你现阶段用不到，你依然可以告诉自己：字典是无序的，这并不影响你的使用。

:::
