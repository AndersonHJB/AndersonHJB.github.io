## Question 1

### 数字分隔与求和

编写一个 C 语言函数，实现以下功能：

1. 从用户那里接受一个不超过 10 位的正整数。
2. 使用函数对该数字进行分隔，输出每一位数字。
3. 对这些数字求和并输出。

### 函数原型

```c
void processAndSumDigits(long num);
```

### 示例

输入：
```
12345
```

输出：
```
Digits: 1 2 3 4 5
Sum: 15
```

### 提示

你可以使用循环来遍历整数的每一位，并进行相应的运算。
