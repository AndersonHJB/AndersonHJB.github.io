## Question 1

::: tip 提问

**我现在有一些困惑，就是编程是每天多多久合适，有没有时间限制，或者你建议每天怎学比较好？**

:::

所有的计划是赶不上变化的，其实我觉得，不是每天学多久合适。在学多久这个问题上，要先解决：怎么学？学什么？

每天自学？还是跟我上课学？

- **自学**
    - 会产生的问题：进度、难度都会拔高～
    - 书：IT 和其他学科不太一样的地方是：除非很基础的内容，这样的内容是不会过时，但是伴随的问题就是除非小孩逻辑性、阅读、英语等本身就很好，这样看书的话是 **<span style="color: orange">有可能学会</span>** 的。
    - 机构网课：机构网课的初衷「大部分」：赚钱，很少是为了让小孩学会，基本上是把课时拉长。

说些掏心掏肺的想法💡吧：很多孩子都是普通孩子，机构「包括视频号、抖音」之所以有成果，走的是**量**还有就是制造家长焦虑「贩卖焦虑」。

**为什么说是走量呢？**

其一：因为，很多是普通的孩子，这些机构的成果只是少部分小孩，稍微一带就成了。

其二：并不是说其他小孩不聪明，还有一些小孩是非常努力💪学好的，这其中除了课程「机构课程，大部分是水课，缺少 **<span style="color:orange">引导</span>** 。

当然，这方面是个大的辩题，需要很多讨论与思考。

这回，再来想想：每天怎么学？目标是什么？——是兴趣？还是什么呢？

最好小孩的内在动力在，肯自主研究🧐，因为如果我们计划好的话，不一定适合小孩。这个也需要一起讨论，你也可以和小孩好好沟通。看看小孩啥想法💡。



## Question 2

::: tip 提问

**从您教学角度看，项目式学习有什么难度？家长该如何配合孩子？**

:::

目前的难度：

- 英语：这个倒不是说，要有多好，主要是要主动遇见英语去记，看见不懂的用有道翻译翻译；先培养习惯。
- 逻辑「长期培养的话，是可以培养起来的」
- 兴趣是有，但是主动性、坚持✊啥的，还是需要多提升。
- 有可能有点容易怕累，这个还是得我们一起适当的去引导。

需要夸、需要有趣，家长可以多和小孩沟通，不一定非要以长辈的身份，可以是朋友🧑‍🤝‍🧑这样。如果怕后面不好管，可以例如：慈母严父、慈父严母都可以。这只是我的一些小想法，这个也需要我们多沟通。

项目式学习，难度就是没基础，英语有些不足「上面说了，不需要太好」虽然进度比较缓慢，但是小孩可以坚持下去且有兴趣，也能学会，具体周期不可预估。**当然，这些都不是绝对的。**

项目式，适合长期坚持和兴趣，虽然不是系统学语法，但是还是可以学到的。当他自己想去了解具体的语法的时候会去深入，这样就有了主动性了。

**玩中学，是对的。**

家长，多和小孩交流「有趣的交流」多「探讨」这样小孩会更积极。

时间关系，有些有可能欠考虑或者不够仔细，我会多思考思考，也期待你的想法。
