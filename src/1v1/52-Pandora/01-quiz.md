## 1. 选择题 (每题 2 分, 共 10 分)

1. Python 的创始人是谁? 

    A. Guido van Rossum

    B. Brendan Eich

    C. Linus Torvalds

    D. Larry Wall  

2. 下列哪个不是 Python 的数据类型? 

    A. Tuple 

    B. Array 

    C. List 

    D. Dictionary  

3. Python中如何表示一个单行注释? 

    A. `#`

    B. `//`

    C. `/* */` 

    D. `--`

4. 以下哪个函数可以用于输出内容? 

    A. print() 

    B. echo() 

    C. out() 

    D. display()  

5. 以下哪种命令可以用于安装 Python 包?

    A. npm install

    B. python setup

    C. pip install

    D. node install  



## 2. 填空题 (每空 2 分, 共 10 分) 

1. 在 Python 中，\_\_\_\_\__ 函数可以用来获取用户输入。
2. Python 中的无穷大可以用 \_\_\_\_\__ 表示。
3. 在 Python 中，字符串是 \_\_\_\_\____ (可变的/不可变的)。 __
4. \_\_\_\_\__ 语句在 Python 中用于异常处理。
5. 在Python中, \_\_\_\_\__ 循环用于遍历序列的元素。



## 3. 编程题 (共 80 分) 

1. 编写一个函数，接受一个整数列表并返回它的最大值和最小值。(20分) 
2. 使用循环，输出 1 到 10 之间的奇数。(10分) 
3. 编写一个函数来计算给定字符串中的元音数量。(20分) 
4. 编写一个 Python 程序来反转一个字符串。(10分) 
5. 使用递归编写一个函数来计算 n 的阶乘。(20分)



## 4. 标准

**得分与 Python 掌握情况**

- **90-100 分**:  Python 高级使用者，对基础概念非常清楚，具有深入的知识。
- **70-89 分**: Python 中级使用者，对大部分基础概念都很熟悉，有一定的实践经验。
- **50-69 分**: Python 初级使用者，了解基本的 Python 语法和特性，但可能对某些高级特性不太熟悉。
- **30-49 分**: Python 入门者，对基本语法有所了解，但仍然需要进一步的学习和实践。
- **0-29 分**: 初学者，需要更多的时间去学习和理解 Python 的基础概念。

这套题目涵盖了 Python 的基础知识，但为了更深入的摸底测试，可以添加更多涉及到数据结构、算法和库的应用的问题。
