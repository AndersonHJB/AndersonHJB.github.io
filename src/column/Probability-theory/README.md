## 课程亮点

1. 只要会四则运算，就能学这门概率论课。用通识的视角，让你的认知水平和概率专家对齐。 
2. 概率论是数学里很基础、很年轻、应用很广泛的一门学科，不仅和我们日常生活息息相关，更是当今大火的大数据和人工智能技术的基础。这门课带你一次性了解概率论的全貌。 
3. 从此，正态分布、幂律分布、大数定律、贝叶斯计算、方差和期望，再也不是高深莫测的数学概念，而是你能运用于自己决策的数学工具。
4. 培养概率思维，提升决策能力。价值投资的本质是什么？保险公司和赌场为什么能赚钱？怎样报考胜算更大？为什么金融危机和地震无法预测？没有信息时怎么提高决策正确率？各种人生难题，用概率论的视角为你重新拆解。
