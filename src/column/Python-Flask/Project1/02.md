## 到底什么是 Flask

你好，我是悦创。

Flask 到底是什么东西呢？

Flask 是一个使用 Python 编写的轻量级 Web 应用框架。其 WSGI 工具箱采用 Werkzeug ，模板引擎则使用 Jinja2 。Flask 使用 BSD 授权。

## Demo

1. 新建文件夹 flask_demo
2. 进入文件夹
3. 新建虚拟环境

::: code-tabs#python

@tab MacOS/Linux

```python
$ mkdir flask_demo
$ cd flask_demo
$ python3 -m venv venv
```

@tab Windows

```python
> mkdir flask_demo
> cd flask_demo
> py -3 -m venv venv
```

@tab 我的实操

```python

```

:::

## 创建 app.py

::: code-tabs

@tab MacOS/Linux

## 一个最小应用

```python
from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello():
    return "Hello AIYC."
	# return "<p>Hello, AIYC.</p>"
```

## 运行 Flask

```python
flask --app app.py run
```

