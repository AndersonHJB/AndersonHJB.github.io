## 1. 什么是 Matplotlib？

Matplotlib 是一个 Python 作图库。它是对数值计算 NumPy 库的一个补充。Matplotlib 是一个强大且广泛使用的库，它提供了一个面向对象的 API，用于将绘图嵌入到通用 GUI 工具箱（如 Tkinter、wxPython、Qt 或 GTK+）中。

## 2. 欢迎来到本课程

欢迎阅读《使用 Matplotlib for Python 创建和定制图表》!在本课程中，我们将帮助你了解 Matplotlib 的基础知识，包括如何使用编程语言通过创建图来直观地表示数据。我们将首先学习如何操作一个图的各个元素，然后再继续研究我们可以用 Matplotlib 绘制的许多不同类型的图。

## 3. 目标受众

本课程是为那些想学习各种可视化数据表示方法的程序员设计的。它是为初学者设计的。

## 4. 推荐的先决条件

尽管本课程的设计是为了让那些不熟悉 Python 的人也能理解，但我们发现，对以下概念的背景了解可以帮助提高理解。

- 对 Python 编程有一定了解。

- 对常见的 Python 库有一定了解，如 NumPy 和 Pandas。

- 一些基本的数学知识。

当然上述情况都不是必要的。

### 4.1 课程的学习成果

通过完成这一课程，你将学会：

- 绘制图表，单图或多个图。
- 设置绘图的标尺、文本、图例和注解元素。
- 自定义绘图的网格。
- 为图片不同元素设置颜色
- 自定义不同组件的风格和外观。
- 根据数据类型和要求选择不同的图表类型。
- 显示数据的分布。
- 显示不同变量之间的关系。
- 以二维和三维方式绘制数据。
