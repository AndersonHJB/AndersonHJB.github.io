## 课程概要

对于数据科学，Matplotlib 是流行的可视化数据的工具之一。 还有很多其他工具，但对于 Python 用户来说，Matplotlib 是必须知道的。

在本课程中，我们将学习如何以几种不同的方式直观地表示数据。 你将学习如何使用图形(figure)和轴(axe)来绘制图表，以及如何从多种类型的对象和模块中进行绘图。

你还将发现控制轴线(spine)的方法以及如何使用 GridSpec 为图形创建复杂的布局，从而创建视觉上令人惊叹的图表。

在课程的后半部分，我们将专注于如何绘制各种类型的图，无论是线条图、饼图还是热图。 总的来说，这是创建有说服力且实用数据图表的简单介绍。

学完之后，相信你将拥有一项重要的新技能。 正如任何数据科学家都知道的那样，必须能够从纷繁复杂的数据中发现规律且可视化出来，才能让人信服。
