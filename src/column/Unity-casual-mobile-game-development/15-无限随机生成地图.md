## 2. 如何在代码中实现

```cs {7}
// filename: PlayerController.cs
if (hit.collider.CompareTag("Wood"))
{
    // TODO: 跟随木板移动
    Debug.Log("在木板上");
    // 所以最简单的是让青蛙跟随父级物体
    transform.parent = hit.collider.transform;  // 把青蛙的父级物体设为当前的碰撞，而我们上面已经判断了碰撞物体 Wood
}
```
