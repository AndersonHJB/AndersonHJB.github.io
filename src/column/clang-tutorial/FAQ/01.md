```c
#include <stdio.h>

int main() {
    double n;
    scanf("%lf",&n);
    if(n >= 0){

       // int a = n + 0.5;
        n = n + 0.5;
        printf("%d",n);
    }else{
        //int b = n - 0.5;
        n = n - 0.5;
        printf("%d",n);
    }

    //printf("%.0f",n);
    return 0;
}

// 输入的任何小数，都出错。如 1.6
```

在C语言中，`%d`是用于输出整数，而不是浮点数。此外，当将浮点数转换为整数时，C语言会将其向下取整为最接近的整数，而不是四舍五入。因此，需要先将浮点数加上或减去0.5，然后再强制转换为整数。

下面是修改后的代码：

```c
#include <stdio.h>

int main() {
    double n;
    scanf("%lf", &n);
    if (n >= 0) {
        int a = (int)(n + 0.5);
        printf("%d", a);
    } else {
        int b = (int)(n - 0.5);
        printf("%d", b);
    }
    return 0;
}
```

在这个修改后的代码中，我们使用了强制类型转换将浮点数转换为整数，同时将其四舍五入到最近的整数。
