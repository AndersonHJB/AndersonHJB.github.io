## 描述

小悦刚刚出生，嗷嗷待哺，一开始他只能学说简单的数字，你跟他说一个整数，他立刻就能学会。

输入一个整数，输出这个整数。

### 输入描述：

输入一个整数，范围在32位有符号整数范围内

### 输出描述：

输出这个整数

## 示例1

输入：

```c
3
```

输出：

```c
3
```



::: details 答案

```c
#include <stdio.h>

int main(void) {
    int i;
    scanf("%d", &i);
    printf("%d", i);
    return 0;
}
```
