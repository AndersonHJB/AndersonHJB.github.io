## 描述

小悦从键盘上输入三个整数，并尝试在屏幕上显示第二个整数。

### 输入描述：

一行输入 3 个整数，用空格隔开。

### 输出描述：

请输出第二个整数的值。

## 示例1

输入：

```c
1 2 3
```

输出：

```c
2
```



::: details 答案

```c
#include <stdio.h>

int main() {
    int smb[3] = {0};
    scanf("%d %d %d", &smb[0], &smb[1], &smb[2]);
    printf("%d", smb[1]);
    return 0;
}
```

```c
#include "stdio.h"

int main() {
    int a, b, c;
    scanf("%d,%d,%d", &a, &b, &c);
    printf("%d", b);
    return 0;
}

// ---output---
1,2,3
2
```

```c
#include "stdio.h"

int main() {
    int a, b, c;
    scanf("%d %d %d", &a, &b, &c);
    printf("%d", b);
    return 0;
}

// ---output---
1 2 3
2
```
