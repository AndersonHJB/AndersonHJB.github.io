## 1. 食材

- [x] 排骨
- [x] 姜片
- [x] 葱段
- [x] 料酒
- [x] 年糕
- [x] 冰糖
- [x] 生抽2勺、老抽1勺、蚝油1勺、白砂糖4勺、辣酱油4勺、蕃茄酱2勺、五香粉1勺、1碗水



## 2. 步骤

1. 排骨切块洗净，冷水下锅，放入姜片3～4片，葱段5～7段，倒入少许料酒去腥，开大火煮沸，加盖约 20 分钟。
2. 把焯好水排骨放入冰水中浸泡 3 分钟，这样方便脱骨「特别说明：有些排骨皮比较薄，若真拆不了，就放弃吧！直接把年糕和排骨一起煮了」。
3. 准备比较有韧性的年糕，这种煮的时候不容易断，把年糕分别塞进排骨肉里。
4. 锅里直接倒油，放入适量冰糖小火炒到全部融化，加少许水稀释，再倒入组装好的排骨，翻炒均匀裹上一层糖色。
5. 灵魂酱汁：生抽2勺、老抽1勺、蚝油1勺、白砂糖4勺、辣酱油4勺（伍斯特辣酱油又称李派林喼汁）、番茄酱2勺、五香粉1勺、1碗水搅拌均匀。
6. 盖上盖子，中小火焖煮5分钟，开盖大火收汁即可出锅。
