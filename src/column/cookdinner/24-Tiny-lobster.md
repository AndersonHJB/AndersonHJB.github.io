## 1. 食材

| 食材名称   | 重量   |
| ---------- | ------ |
| 小龙虾🦞    | 2斤    |
| 熟白芝麻   | 少许   |
| 油         | 适量   |
| 盐         | 少许   |
| 郫县豆瓣酱 | 2汤勺  |
| 生抽       | 2汤勺  |
| 蚝油       | 1汤勺  |
| 米酒       | 2汤勺  |
| 茴香       | 半汤勺 |
| 花椒       | 半汤勺 |
| 桂皮       | 少许   |
| 冰糖       | 10颗   |
| 干辣椒     | 20个   |
| 大葱       | 1小节  |
| 蒜         | 10瓣   |
| 姜         | 1小块  |
| 香菜       | 1棵    |
| 十三香     | 1茶匙  |

::: tip 酱香和麻辣的区别

酱香按原本的，麻辣的需要把干辣椒掰碎，然后可以考虑加辣椒粉。

:::

## 小贴士

1、龙虾买回来用清水多养，中途多换几次水，让龙虾吐出脏物。

2、龙虾焖熟后，汁别收太干了，盛入容器时，尽量有少许汤汁浸着龙虾，这样更好吃有味哦。
