## 2. SQL

上节课我们了解到，我们「APP」操作 DBMS「数据库管理系统」。那肯定使用的是接口吧，接口肯定需要通信语言。这个通信语言肯定是各种各样的，在我们的数据库中叫做：SQL。

- 结构化的查询语言「**Structured Query Language** (SQL) 」
- 是与 DBMS、关系型数据库「Relational Database」通信的标准语言

那有关系型数据库，就会有非关系型数据库。
