## 1. 通过命令行安装

无论是 Mac 还是 Windows，都有两种命令行。一种是系统自带的命令行，另一种是 Pycharm 命令行。

这里，主要带你使用 Pycharm 命令行来安装，比较通用。

### 1.1 打开 Pycharm

![](./FAQ1-python-installation-library-methods.assets/image-20240321152253984.png)

### 1.2 使用 pip 安装

比如我们这里需要安装 numpy 库，我们则使用如下命令：

```bash
pip install numpy
# pip install 你需要安装的库名称
```

### 1.3 在 Terminal 执行命令

![](./FAQ1-python-installation-library-methods.assets/image-20240321153350582.png)

## 2. 通过 Pycharm 直接安装

### 2.1 随便一个 Python 代码文件

随便一个 Python 的代码文件中，导入你所需要的库，不论是否安装。

![](./FAQ1-python-installation-library-methods.assets/image-20240321155318698.png)

### 2.2 开始安装

鼠标点击 numpy 库，然后按住键盘 Alt 键 + Enter。
