要求：在空间复杂度 O(1) 的情况下，时间复杂度最小。

```python
class Solution:
    def hasCycle(self, head: ListNode) -> bool:
```

## Sample Input

```python
A -> B -> C -> D -> B -> C -> D
```

## Sample Output

```python
True
```
