自然数取余定义分为两种：

  1. 定义1：如果 a 和 d 是两个自然数，d 非零，可以证明存在两个唯一的整数 q 和 r，满足 a=qd+r 且 0 ≤ r < d（其中 q 为商，r 为余数）。
  2. 定义1一般作为数学中的取余法则，即两个数取余，余数总是为正数。
